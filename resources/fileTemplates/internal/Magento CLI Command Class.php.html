Magento enables your component to add commands to our Symfony-like command-line interface (CLI). Read more about Magento CLI commands.
Template's variables:
${NAMESPACE}   Created PHP CLI command class namespace.
${NAME}   PHP class for the CLI command name.
${NAME}   PHP class for the CLI command name.
${CLI_COMMAND_NAME}   Name of the CLI used under the bin/magento.
${CLI_COMMAND_DESCRIPTION}   Description of the CLI command used under the bin/magento.
